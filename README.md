<h1 align="center">📊 Data Analytics Bootcamp | Proyecto Final End-to-End</h1>

<hr>

<h2>📌 Resumen</h2>

<p>
El siguiente trabajo corresponde a la <strong>entrega final de un Bootcamp orientado a Data Analytics</strong>. 
Se trata de un desarrollo integral que abarca todo el ciclo de vida del dato: desde la <strong>obtención y procesamiento de información</strong> hasta la <strong>construcción de modelos de Machine Learning</strong> para la toma de decisiones basada en datos.
</p>

<p>
El proyecto se desarrolló a partir de un conjunto de archivos <strong>.csv provenientes de Kaggle</strong>, a partir de los cuales se diseñó un <strong>modelo entidad-relación</strong> que estructuró la construcción de la base de datos tanto en <strong>Google BigQuery</strong> como en <strong>Power BI</strong>.
</p>

<p>
Se realizó una presentación inicial del proyecto en <strong>PowerPoint</strong> con el objetivo de obtener una aprobación preliminar por parte de los Stakeholders vinculados.
</p>

<p>
El resultado final fue presentado mediante un <strong>Dashboard Interactivo</strong> y defendido en vivo ante un auditorio, incluyendo la explicación técnica del modelo analítico y el algoritmo de aprendizaje automático implementado.
</p>

<hr>

<h2>🛠️ Skills Técnicas Aplicadas</h2>

<h3>📥 Obtención y Comprensión de Datos</h3>
<ul>
  <li>Extracción de datos desde Kaggle (.csv)</li>
  <li>Análisis exploratorio de datos (EDA)</li>
  <li>Diseño del modelo entidad-relación</li>
</ul>

<h3>🧼 Limpieza y Transformación de Datos</h3>
<ul>
  <li>Uso de <strong>Python</strong> para limpieza y normalización</li>
  <li>Transformación de archivos originales en tablas finales estructuradas</li>
  <li>Tratamiento de valores nulos, inconsistencias y duplicados</li>
</ul>

<h3>☁️ Ingeniería de Datos en la Nube</h3>
<ul>
  <li>Construcción de base de datos en <strong>Google BigQuery</strong></li>
  <li>Automatización de carga de datos hacia Google Cloud</li>
  <li>Documentación detallada del flujo automatizado mediante videos explicativos</li>
</ul>

<h3>🧠 Machine Learning</h3>
<ul>
  <li>Implementación de modelo supervisado con <strong>Scikit-Learn</strong></li>
  <li>Aplicación de <strong>árboles de decisión</strong></li>
  <li>Diseño de algoritmo orientado a responder una pregunta de negocio concreta</li>
  <li>Evaluación del modelo para soporte en toma de decisiones</li>
</ul>

<h3>📊 Business Intelligence</h3>
<ul>
  <li>Modelado de datos en <strong>Power BI</strong></li>
  <li>Construcción de Dashboard Interactivo</li>
  <li>Visualización estratégica para análisis y toma de decisiones</li>
</ul>

<h3>🎤 Comunicación y Presentación</h3>
<ul>
  <li>Presentación inicial para validación con Stakeholders</li>
  <li>Defensa final en vivo ante audiencia</li>
  <li>Explicación integral del pipeline de datos y modelo predictivo</li>
</ul>
